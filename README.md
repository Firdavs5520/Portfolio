# Firdavs Portfolio

Oddiy statik portfolio sahifasi. Ochish uchun `index.html` faylini browserda ishga tushiring.

Tahrirlash kerak bo'ladigan asosiy joylar:

- `index.html` ichida ism, kasb, loyiha matnlari, email va Telegram manzili.
- `styles.css` ichida ranglar, bo'shliqlar va responsiv dizayn.
- `assets/portfolio-hero.png` hero fon rasmi.

Preview screenshotlar `artifacts` papkasida saqlangan.
